## 7. 建议和最佳实践

### ✅ 当前最佳实践

1. **三重信息编码**
   - 颜色 + 文本 + 图标表达同一信息
   - 示例：危险按钮 = 红色 + "删除" 文本 + ✕ 图标

2. **高对比度默认值**
   - 所有关键文本使用 WCAG AAA 级对比度
   - 仅装饰元素可降低到 AA

3. **清晰的 Focus 状态**
   - 所有交互元素都可通过键盘访问
   - Focus 指示器满足对比度要求

4. **渐进式增强**
   - 基础功能不依赖颜色
   - 辅助视觉提示增加可用性

### 🚫 应避免的做法

- ❌ 不要使用红/绿组合（色盲用户）→ **已避免**
- ❌ 不要仅用颜色表达重要信息 → **已避免**
- ❌ 不要使用对比度 < 3:1 的互动元素 → **已避免**
- ❌ 不要隐藏 focus 指示器 → **已避免**
